## PSI
##### Output number of PSIs in the given integer array.
```
PSI - positive sum interval - consecutive subsequence where sum of all elements in the subsequence > 0

sample input: 1 -2 5 -7 4 3
sample PSI: {1, -2, 5}, {-2, 5}, {5, -7, 4, 3}
sample output: 12
```
